In this project, the process of obtaining a new application on a monthly or annual basis is provided.

In index html, the steps are the same, we designed the right side in a different way.

As we know, we have divided CSS into font, btn, color, reset, media, and styles to give properties to each element.

We did the operations we would do in main js. First, the user has to go through the form, after specifying his identity there, he has to specify whether he wants to receive the package monthly or annually. At the end, the values of the packages he has chosen are added up and the number is shown to us.
